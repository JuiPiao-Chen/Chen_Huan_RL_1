# Chen_Huan_RL_1

A personal webpage project for the Deep Reinforcement Learning course.

## Author

**CHEN, JUI-PIAO**

## Project Description

This project is a modern personal webpage featuring a cyberpunk-style design with dynamic elements including real-time date and time display.

## Features

- **Cyberpunk Design**: Dark background with neon cyan glow effects
- **Dynamic Grid Background**: 3D perspective animated grid
- **Floating Particles**: Randomly floating light particles
- **Scanline Effect**: Retro CRT monitor aesthetic
- **Light Beam Animation**: Vertical moving light beams
- **Glowing Border**: Gradient breathing light effect
- **Real-time Clock**: Live updating date and time display
- **Responsive Layout**: Works on different screen sizes

## Technologies Used

- HTML5
- CSS3 (Animations, Flexbox, Gradients)
- JavaScript (Date/Time, DOM manipulation)
